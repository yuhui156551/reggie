# reggie
## 外卖项目

### 技术栈

  SpringBoot2、MybatisPlus、AspectJ、MySQL、Lombok、Knife4j。
  
### 实现
  
  主要实现功能：选择外卖、创建订单、后台管理员工、管理菜单、管理订单等。
